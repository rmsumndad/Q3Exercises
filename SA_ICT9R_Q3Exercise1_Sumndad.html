My First Bootstrap Page



HTML

HyperText Markup Language

This is the beginning of making a web page. In an html file, it uses the .html at the end of the file name to make the html file work. In the file you can add images, texts, tables, links and etc. You can also design the html file with certain tags, attributes and codes.

CSS

Cascasing Style Sheet

This Can help you modify the codes of the HTML tag. This can be the size, the position, the color of the text and background, margins, and etc. There are 3 types of CSS which is the Internal, Inline and External.

Bootstrap

Bootstrap Version 4

This was first used with Twitter by @mdo and @fat, which was also known as the Twitter Blueprint. With this you can also use HTML and CSS codes. There are four types of containers which are the container-sm, container-md, container-lg, container-xl.
